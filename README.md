# SHL GenAI Assessment Recommendation System

This project builds a GenAI-powered recommendation engine that maps job descriptions to relevant SHL assessments using embeddings, FAISS, and FastAPI.

## Features
- SHL catalog crawler
- Semantic search with embeddings
- Balanced K & P recommendations
- Recall@10 evaluation
- API + Web UI

## Run
1. Crawl data
2. Build embeddings
3. Start API
4. Launch Streamlit
